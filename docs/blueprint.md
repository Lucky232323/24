# **App Name**: RIDER APP

## Core Features:

- User Authentication: Secure mobile number-based login/signup with OTP verification and optional social login.
- Real-time Map Display: Display user's location and nearby vehicles on a live map interface. The driver/rider heading will be calculated every second.
- Smart Dispatch Tool: AI tool algorithm matches captains to riders based on proximity, historical data, and reliability scores.
- Fare Calculation: Calculates fare estimation based on route and selected service with real-time updating
- Real-time Trip Tracking: Animated vehicle icon updates every 3-5 seconds on the map to the destination, and includes verification code before boarding vehicle.
- Route Optimization Tool: Integrate advanced routing that considers real-time traffic and known choke points, provides fastest routes using AI as a tool.
- Rating and Feedback: Collect 5-star ratings and feedback using preset feedback.

## Style Guidelines:

- Primary color: Rapido Yellow/Saffron (#F9C935) for CTAs and primary actions.
- Accent color: Red (#FF4040) for alerts, warnings, and the SOS button.
- Background color: Light gray (#FAFAFA) to maintain contrast and readability.
- Body and headline font: 'Inter', a sans-serif font, providing a modern user interface. Note: currently only Google Fonts are supported.
- Use touch targets of at least 48dp x 48dp for all buttons and icons.
- Map view should occupy 70-80% of the Home Screen, buttons will take height of 56dp, and searchbar is to take rest of height.
